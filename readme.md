# Qwen3-Embedding-0.6B-CoreML-conversion
Qwen3 Embedding 0.6B converted to CoreML .mlpackage

TestApp XCode project for iOS contains example usage of CoreML framework with the model.
Check EmbedModel.swift in TestApp Project folder for usage.
- The model file, named as 'Qwen3Embedding06B.mlpackage' needed to be placed in same folder as TestApp.swift for app to run without any additional edits.

Check Python-code-used-for-conversion/ to convert the model.
(Using conda environment is recommended.)

[Model Huggingface page](https://huggingface.co/Qwen/Qwen3-Embedding-0.6B)
